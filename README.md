# Tagirov Digital Studio

Digital-студия: **сайты и платформы**, **CRM и автоматизация**, **UX/UI**, **бренд**. Сайт на Next.js с динамическим портфолио и админ-панелью под ваш контент.

---

## Для заказчиков: почему нам можно доверять

Ниже — не «маркетинг ради маркетинга», а опоры, на которых обычно строится спокойное решение о сотрудничестве.

### 1. Прозрачный процесс

- **Диагностика и цели** — фиксируем, что считаем успехом проекта.
- **Понятные этапы** — от концепции до запуска и поддержки, без «чёрного ящика».
- **Регулярная связь** — вы в курсе статуса, сроков и рисков.

### 2. Опора на факты, а не на обещания

- **Живые кейсы** в разделе портфолио — реальные задачи, подход и результат (там, где это уместно раскрывать).
- **Чёткое описание услуг** — что входит, на чём специализация, без размытого «всё и сразу».

### 3. Юридическая и финансовая предсказуемость

- **Договор и ТЗ** — объём работ, сроки, порядок согласований (по договорённости).
- **Поэтапная оплата** при классической модели — оплата привязана к этапам, а не к «всё вперёд».

### 4. Техническая ответственность

- **Современный стек** — быстрый и поддерживаемый фронтенд, отдельный API под данные и формы.
- **Ваш контент под контролем** — кейсы и заявки можно вести через админ-интерфейс без правки кода «вручную» каждый раз.

### 5. Безопасный первый шаг

- **Консультация или короткий созвон** — разбор задачи без обязательства «купить сегодня».
- **Заявка с сайта** — оставьте имя и удобный контакт (email, телефон или Telegram); комментарий по желанию.

### Что можно добавить на сайт позже (ещё сильнее доверие)

Если решите развивать витрину под конверсию, имеет смысл поэтапно добавить:

| Элемент | Зачем |
|--------|--------|
| **Отзывы с именем и компанией** (и по возможности ссылкой/логотипом) | Социальное доказательство «люди уже работали». |
| **Логотипы клиентов** | Быстрая визуальная оценка опыта. |
| **Цифры** (сроки внедрения, объём автоматизации, рост метрик — только честно) | Конкретика вместо общих фраз. |
| **Блок «Как мы работаем»** в 4–6 шагов | Снижает страх «не знаю, что будет дальше». |
| **FAQ** по договору, срокам, правкам | Меньше скрытых вопросов — больше заявок. |
| **Сертификаты / партнёрства** (если есть) | Дополнительный знак качества. |

Их можно встроить в существующие страницы или вынести отдельным блоком на главную — без ломки текущей архитектуры.

## Для разработчиков и владельца проекта

**Стек:** Next.js 15 · Tailwind · Framer Motion · Three.js (оптимизированный hero) · Django (API, портфолио, заявки).

**Hero (≥1024px):** сцена «AI-ядро» — оболочка узлов, линии к центру, «зонды», parallax от мыши, только `MeshBasicMaterial` / линии, без post-processing. **&lt;1024px:** лёгкий CSS/SVG fallback.

```bash
npm install
npm run dev
```

**Backend и портфолио:** см. [RUN_LOCAL.md](./RUN_LOCAL.md).

**Публикация на сервер:** см. [DEPLOY_GUIDE.md](./DEPLOY_GUIDE.md).

Зависимости hero: `three`, `@react-three/fiber`.

**Логотип:** файл `public/LogoWhite.png` (копия из `RobertFile/LogoWhite.png`) — шапка и подвал.
